count-ui

A lightweight, HTML-first web component for displaying formatted numbers with separators, short notation, prefixes, suffixes, and precision.

See Examples

Easy to Use

Just add <count-ui val="1234"> and it works. No framework required.

Flexible Formatting

Thousands separators, K/M/B short notation, decimal precision, prefix & suffix support.

Lightweight & Fast

Pure web component with zero dependencies. Small, fast, and easy to integrate.
